# Enterprise-IT-Infastrucutre-And-Database-Design

Overview

This project demonstrates the design and implementation of a simulated enterprise IT infrastructure. The goal was to design a scalable, secure, and reliable network environment that supports business operations across multiple departments.

The infrastructure was designed using Cisco networking concepts, including VLAN segmentation, dynamic routing, and secure access to internal services. The project also focuses on network organization, reliability, and efficiency.

This project was completed as part of my Computer Information Technology coursework and serves as an example of my ability to design and implement real-world network infrastructure.
